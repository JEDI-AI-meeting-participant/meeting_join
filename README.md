# JEDI Realtime Websocket

This is a realtime websocket application built with React, Vite, and TypeScript.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repo
2. Install NPM packages

### Running the Application

```sh
npm run start
```
